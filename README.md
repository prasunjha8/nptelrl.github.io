## What's Inside

These notes are structured as a web portal with a page per week. Each page covers:

- **Core theory** with full mathematical derivations
- **SVG diagrams** for algorithms and architectures
- **Every practice & graded assignment question** with solutions and explanations

Topics covered include value iteration, policy iteration, Monte Carlo methods, TD learning, SARSA, Q-learning, eligibility traces, linear and deep function approximation (DQN), policy gradient methods (REINFORCE, Actor-Critic), average reward formulation, hierarchical RL (SMDPs, Options, HAMs, MAXQ), and option discovery.
