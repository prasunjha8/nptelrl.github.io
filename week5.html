Reinforcement Learning Examination Notes

These notes cover core algorithms for finding optimal policies and value functions in MDPs: Value Iteration, Policy Iteration, Asynchronous DP, and Monte Carlo methods.

1. Value Iteration and Contraction Mappings

Value iteration involves successively updating an estimated value function v until it converges to the optimal value function v*.

The Bellman Optimality Operator (L)

(Lv)(s) = max_{a ∈ A} [ R(s,a) + γ Σ P(s' | s,a) v(s') ]

The space of all bounded value functions forms a complete Banach space under the max norm.

  • L is a contraction mapping with factor γ < 1
  • L has a unique fixed point v*
  • L is non-linear due to the max operation

Mathematical Bounds

||Lv - Lu|| ≤ γ ||v - u|| |Lv(s) - Lu(s)| ≤ γ ||v - u||

3. Dynamic Programming (DP) Variants & GPI

Classic DP requires full model knowledge.

Asynchronous DP

Updates states in any order; converges if all states are updated.

RTDP

Focuses updates along sampled trajectories.

Generalized Policy Iteration (GPI)

  • Policy evaluation and improvement interact continuously
  • No need to wait for full evaluation
  • Converges when policy is greedy w.r.t its value
