Value Iteration and Contraction Mappings

Value iteration updates a value function until it converges to the optimal value function.

Bellman Optimality Operator

(Lv)(s) = max_a [ R(s,a) + γ Σ P(s'|s,a) v(s') ]

The operator is a contraction mapping with factor γ < 1 and has a unique fixed point.

Stopping Criteria

|| v(n+1) - v(n) || < ε(1 - γ) / (2γ)

This guarantees near-optimal value and policy.
