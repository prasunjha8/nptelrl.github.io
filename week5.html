1. Value Iteration and Contraction Mappings

Value iteration involves successively updating an estimated value function until it converges to the optimal value function.

Bellman Optimality Operator

(Lv)(s) = max_a [ R(s,a) + γ Σ P(s' | s,a) v(s') ]

The space of value functions forms a Banach space under the max norm.

  • Operator L is a contraction with factor γ < 1
  • It has a unique fixed point v*
  • It is non-linear due to max operator

Mathematical Bounds

||Lv - Lu|| ≤ γ ||v - u|| |Lv(s) - Lu(s)| ≤ γ ||v - u||

3. Dynamic Programming Variants

Classic DP requires full environment model.

Asynchronous DP

Updates states in any order and converges if all states are visited.

RTDP

Focuses only on sampled trajectories.

Generalized Policy Iteration (GPI)

  • Evaluation and improvement interact continuously
  • No need for full evaluation before improvement
  • Converges when policy becomes greedy with respect to its value

4. Monte Carlo Methods

Used when transition model is unknown.

  • Learn from episodes
  • Updates happen at episode end
Example: All visited states in trajectory can be updated.

First Visit vs Every Visit

  • First Visit: use first occurrence
  • Every Visit: use all occurrences
